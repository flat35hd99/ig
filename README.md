## IG

Download source file, Unpack, Build, Install, and create modulefile

### Usage

```console
ig install nvim

> input where is source
GitHub
> input github repo name
neovim/neovim
> select release
NVIM v0.7.0
> select file
nvim-linux64.tar.gz
unpacking...
> Input version
0.7.0
creating modulefile...
Done!
```

### What this can do

- Download a file.
- Unpack the file.
- Install selected directory.
- Create modulefile.
- Save processes and reproduce other place.
